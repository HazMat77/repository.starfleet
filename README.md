# Starfleet Repository

Kodi repository hosting `plugin.video.starfleet`.

## Install in Kodi

1. Go to **Settings → File manager → Add source**
2. Enter the URL: `https://raw.githubusercontent.com/HazMat77/repository.starfleet/main`
3. Name it **Starfleet**
4. Go to **Add-ons → Install from zip file → Starfleet**
5. Open `repository.starfleet/` → install `repository.starfleet-1.0.0.zip`
6. Go to **Add-ons → Install from repository → Starfleet Repository**
7. Install **Starfleet** from Video add-ons
